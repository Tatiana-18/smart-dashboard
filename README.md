# 🦊 Smart Dashboard

**PWA-приложение для повышения личной продуктивности**

## 📋 Описание

Smart Dashboard — модульное веб-приложение для отслеживания задач, заметок и прогресса активности.

## 🏗️ Архитектура

- **Core**: Router, AuthService, DataService, CacheService, NotificationService
- **Modules**: Tasks, Notes, Activity Tracker
- **PWA**: Manifest, Service Worker, Offline режим

## 🚀 Запуск

1. Клонируйте репозиторий
2. Откройте `public/index.html` в браузере
3. Для PWA: задеплойте на GitHub Pages/Render

## 📱 Установка как PWA

1. Откройте в браузере (Chrome/Safari)
2. Нажмите «Добавить на главный экран»
3. Приложение установится как нативное

## 🎨 Особенности

- Адаптивный дизайн (Mobile First)
- Розово-фиолетовая палитра
- Маскот-лисичка 🦊
- Геймификация (баллы, достижения)
